# useful.utils

A personal collection of **small utilities, scripts, tweaks, and workarounds** that I use across different tools, systems, and workflows.

This repository is **not a framework** and **not a polished product**.  
It’s a practical toolbox.

## Purpose

The goal of this repository is simple:

- Increase **efficiency and productivity**
- Fix or mitigate **annoying defaults**, broken behaviors, or missing features
- Reduce repetitive manual work
- Make everyday tools behave *the way I actually want them to*

Most of the content exists because:
- the original tool does not offer the option,
- the setting is hidden, poorly designed, or overly verbose,
- or the workflow is unnecessarily slow.

## What you can find here

Depending on the folder / script:

- Small **helper scripts** (CMD, PowerShell, Bash, JS, Python, etc.)
- Configuration tweaks and overrides
- Minimal automation snippets
- Workarounds for UI/UX annoyances
- Notes or experiments that are useful in practice

The common theme:  
**practical usefulness over elegance**.

## What this repo is NOT

- ❌ A curated library
- ❌ Guaranteed stable APIs
- ❌ Backward compatibility promises
- ❌ Extensive documentation for every script

Some things are rough by design — they solve *my* problem first.

## Usage

- Use anything you find useful
- Copy, modify, adapt freely
- Expect scripts to be opinionated
- Read before running (some tools touch system settings)

If something breaks — that’s on you 🙂

## Structure

The structure may evolve over time.  
Directories are grouped by **problem domain**, not by language or purity.
